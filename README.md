
ТЕРМИНЫ И ОПРЕДЕЛЕНИЯ

Сухая масса ракеты — масса конструкции ракеты без топлива и без полезной нагрузки.

Масса топлива — совокупная масса всех компонентов топлива, расходуемых в процессе полёта.

Расход топлива — количество топлива, сгорающее за единицу времени, кг/с.

Тяга двигателя — сила реактивного двигателя, измеряется в ньютонах (Н).

Ускорение — изменение скорости ракеты, м/с².

Сопротивление воздуха — сила, зависящая от плотности и скорости воздуха.

Телеметрия — данные о параметрах полёта: высота, скорость, масса топлива и др.

Пресет параметров — сохранённый JSON-файл исходных данных.

Симуляция — программная имитация полёта ракеты.

Шаг интегрирования (Δt) — временной шаг расчёта.

JSON — текстовый формат обмена данными.

FPS — частота обновления визуализации.

Режим «Блуждание ракеты» — свободный полёт с изменяемой средой.

Редактор ракеты — модуль изменения конструктивных параметров.

ВВЕДЕНИЕ

В документе изложены требования к разработке программы симуляции запуска ракеты, предназначенной для обучающихся среднего общего и среднего профессионального образования.

Существующие профессиональные инструменты (GMAT, STK) слишком сложны для школьников. Разрабатываемый продукт упрощает изучение баллистики и космических процессов.
Программа поможет обучающимся разобраться с законами движения, параметрами ракеты и моделированием её полёта.
