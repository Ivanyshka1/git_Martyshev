1 ВВЕДЕНИЕ

В документе изложены требования к разработке программы симуляции запуска ракеты, предназначенной для обучающихся среднего общего и среднего профессионального образования.

Существующие профессиональные инструменты (GMAT, STK) слишком сложны для школьников. Разрабатываемый продукт упрощает изучение баллистики и космических процессов.
Программа поможет обучающимся разобраться с законами движения, параметрами ракеты и моделированием её полёта.

3 НАЗНАЧЕНИЕ РАЗРАБОТКИ

Программа создаётся для учебных целей — улучшить понимание физики запуска космических объектов.
